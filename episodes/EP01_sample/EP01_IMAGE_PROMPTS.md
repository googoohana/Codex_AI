# EP01_IMAGE_PROMPTS

## 프롬프트 작성 원칙

모든 이미지 프롬프트는 `EP01_CHARACTER_BIBLE.md`와 `EP01_SHOT_PLAN.md`를 함께 사용합니다.

영문 프롬프트는 이미지 모델이 장면을 명확하게 이해하도록 작성한 제작 지시서입니다. 실제 생성 전에는 사용하는 이미지 모델의 최신 공식 가이드에 맞게 세부 표현을 다시 검수합니다.

## 이미지 생성 전 최종 확인

1. 모든 프롬프트가 대응하는 컷 목적을 설명하는가
2. 캐릭터 고정 문장이 반복되어 있는가
3. 장소와 카메라 정보가 포함되어 있는가
4. `no readable text`, `no logo`, `no extra people` 제약이 포함되어 있는가
5. 참조 자산과 권리 정책이 포함되어 있는가
6. 아직 실제 이미지를 생성하지 않고 검수 상태에서 멈췄는가
