# EP01 참조 자산 안내

이 폴더는 Kling, Seedance, Grok 같은 영상 모델이 캐릭터와 장소를 일관되게 유지하도록 돕는 참조 자료를 저장하는 곳입니다.

## 폴더

```text
characters/
locations/
```

## 저장 원칙

1. 캐릭터 참조 자료는 `characters`에 저장한다.
2. 장소 참조 자료는 `locations`에 저장한다.
3. 파일 이름에는 역할과 버전을 적는다.
4. 실존 인물의 얼굴, 목소리, 사진은 사용 권리를 확인한 뒤 저장한다.
5. Seedance나 Kling에 입력하기 전 최신 공식 제한을 다시 확인한다.

## 파일 이름 예시

```text
CHAR_KIM_JEONGAE_front_v01.png
CHAR_KIM_JEONGAE_side_v01.png
LOC_HOME_LIVINGROOM_v01.png
LOC_HOSPITAL_ROOM_v01.png
```

현재 EP01 샘플은 가상의 캐릭터를 사용하며, 아직 실제 참조 이미지는 생성하지 않았습니다.
