# 달 배경 인물 움직임 테스트 계획

## 현재 상태

- 시작 이미지: `slide_09_moon_bear_woman.jpg` (`1080×1350`)
- Comfy Cloud 작업 흐름: `test_wan_moon_motion_no_run`
- 모델 템플릿: Wan 2.5 이미지에서 비디오
- 표시 비용: 105.5 credits/Run
- 실제 실행: 하지 않음

## 장면 목표

별이 가득한 달 표면 같은 공간에서 큰 곰 캐릭터는 접이식 의자에 앉아 있고,
단발머리 성인 여성은 떨어진 벤치에 앉아 휴대폰을 바라보고 있다.

## 움직임 설계

1. 별빛과 달빛이 아주 미세하게 반짝인다.
2. 큰 곰 캐릭터가 천천히 숨을 쉬고 고개를 여성 쪽으로 조금 돌린다.
3. 곰의 손가락과 스웨터가 아주 미세하게 움직인다.
4. 단발머리 여성이 휴대폰 화면을 한 번 스크롤한 뒤 천천히 무릎 쪽으로 내린다.
5. 여성이 고개를 아주 조금 들어 곰 쪽을 짧게 바라본다.
6. 카메라는 거의 고정하고 매우 느린 줌인만 사용한다.

## 품질 보호 규칙

- 곰 캐릭터와 여성의 얼굴, 몸, 의상을 유지한다.
- 두 인물의 거리와 앉은 위치를 유지한다.
- 벤치, 접이식 의자, 가방, 휴대폰을 유지한다.
- 새로운 인물을 만들지 않는다.
- 갑작스러운 장면 전환을 만들지 않는다.
- 달의 모양을 변형하지 않는다.
- 과도한 카메라 움직임을 사용하지 않는다.

## 멈춤 이유

Comfy Cloud에서 확인한 영상 생성 템플릿은 크레딧을 사용한다.

사용자의 조건은 크레딧을 소모하지 않는 테스트이므로 실제 실행하지 않고 준비 상태에서 멈춘다.
